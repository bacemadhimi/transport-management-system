Politique de sécurité des mots de passe :

  • lock_clock Âge minimum : 1 jour (vous ne pouvez pas changer votre mot de passe avant 24h)
  • update Âge maximum : 42 jours (vous devez changer votre mot de passe tous les 42 jours)
  • rule Longueur minimale : 7 caractères
  • history Historique : Les 24 derniers mots de passe sont mémorisés
  • security Complexité : Activée - Doit contenir majuscule, minuscule, chiffre et caractère spécial
