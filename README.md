\# Transport Management System (TMS)



Projet full-stack composé de :

\- Backend : .NET 9 Web API

\- Frontend : Angular 20

\- Objectif : gestion complète d’un système de transport (clients, chauffeurs, camions, commandes, livraisons, optimisation…)



La documentation, le planning, les releases \& sprints seront placés dans `/docs`.
